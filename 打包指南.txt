
## 发布与打包

如果您需要将程序发送给其他用户，请按照以下步骤生成独立的运行包（绿色版）：

1.  打开 **Qt 命令行终端** (例如 "Qt 5.15.2 (MSVC 2019 64-bit)")。
    *   *注意：不要在普通的 CMD 或 PowerShell 中运行，否则会找不到 qmake 命令。*
2.  进入项目目录：
    ```bash
    cd c:\Users\admin\Desktop\trae\exerciseReminder
    ```
3.  运行打包脚本：
    ```bash
    build_and_deploy.bat
    ```
4.  脚本运行完成后，会在当前目录下生成一个名为 `ExerciseReminder_Dist` 的文件夹。
5.  **测试**：进入该文件夹，双击 `ExerciseReminder.exe` 确保能正常运行。
6.  **分享**：将整个 `ExerciseReminder_Dist` 文件夹压缩成 `.zip` 或 `.7z` 文件，发送给用户即可。用户解压后即可直接运行，无需安装 Qt 环境。
